# Chemistry Equation Solver  
  
A comprehensive tool for solving and balancing chemical equations with both GUI and Telegram bot interfaces.  
  
## Features  
  
- ? **Chemical Equation Balancing** - automatic coefficient finding  
- ?? **Periodic Table** - atomic masses of elements  
- ?? **Solubility Table** - solubility information for salts  
- ?? **Acids and Bases** - classification and properties  
- ?? **Reference Materials** - formulas, valences, indicators  
- ?? **Telegram Bot** - access all features via Telegram  
  
## Requirements  
  
### GUI Version  
- Python 3.6 or higher  
- Standard Python library (tkinter)  
  
### Telegram Bot Version  
- Python 3.6 or higher  
- python-telegram-bot==20.7  
  
## Installation  
  
### GUI Version  
1. Make sure Python 3.6+ is installed  
2. No additional packages required - uses only standard library  
  
### Telegram Bot Version  
1. Install dependencies:  
```bash  
pip install -r requirements.txt  
```  
2. Get bot token from [@BotFather](https://t.me/botfather) on Telegram  
3. Set token in `config.py` or environment variable `TELEGRAM_BOT_TOKEN`  
  
## Running  
  
### GUI Version  
```bash  
python chemical_equation_solver.py  
```  
  
### Telegram Bot Version  
```bash  
python telegram_chemistry_bot.py  
```
